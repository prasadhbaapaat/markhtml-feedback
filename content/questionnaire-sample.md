## Questionnaire Mode

Did you know you can turn any Markdown document into an interactive survey? 
By changing the format to `'questionnaire'` in your `config.php`, the system will automatically turn every Ordered List item into an inline answer text box!

For example, if this document was in Questionnaire Mode, the following list would transform into text boxes:

1. What is your favorite programming language?
2. How often do you use Markdown?
3. Do you have a Screenshot [UPLOAD, 'Upload Screenshot' ]
4. Any other feedback?
